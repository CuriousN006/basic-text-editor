# Basic Text Editor

설치 없이 Edge 또는 Chrome에서 실행하는 오프라인 리치 텍스트 편집기입니다.

Word와 비슷한 방식으로 글을 편집하고, 네이버 카페 같은 웹 편집기에 서식 있는 상태로 복사하는 용도로 만들었습니다.

## 주요 기능

- 단일 `index.html` 파일로 오프라인 실행
- 외부 라이브러리, 서버, 로그인, Node.js 불필요
- 마크다운, HTML, 일반 텍스트 파일 불러오기
- HTML 및 마크다운 저장
- 제목, 굵게, 기울임, 밑줄, 취소선, 목록, 인용문, 코드 블록
- 커서 위치와 입력 상태에 맞춘 서식 버튼 및 문단 종류 표시
- 빈 줄을 편집 가능한 빈 문단으로 보존
- 선택한 글자 위에 URL을 붙여 넣어 링크 생성
- 링크와 팝업 사이의 연결 영역 및 클릭·주소 복사가 가능한 링크 미리보기
- Word처럼 Ctrl+클릭으로 링크 열기
- 드래그한 글자를 자동 입력하는 찾기와 찾아 바꾸기
- Word식 특수기호 치환 (`^p`, `^l`, `^t`, `^^`)
- 외부 서식을 제거하고 원문의 줄바꿈·빈 줄·탭을 항상 보존하는 붙여넣기 (`Ctrl+Shift+V`는 URL 자동 링크도 끔)
- 받침에 맞춘 한국어 조사 자동 교정
- 네이버 카페에서 `Shift+Enter` 줄과 빈 줄이 유지되도록 각 줄을 독립 블록으로 변환해 리치 텍스트 복사

## 실행 방법

1. 저장소에서 `index.html`을 다운로드합니다.
2. 파일을 더블클릭합니다.
3. 연결 프로그램을 묻는 경우 Edge 또는 Chrome을 선택합니다.

자세한 내용은 [사용법](사용법.md)을 참고하세요.

## 네이버 카페에 붙여 넣기

1. 문서를 작성하거나 파일을 불러옵니다.
2. **전체 서식 복사**를 누릅니다.
3. 네이버 카페 글쓰기 창에서 `Ctrl+V`를 누릅니다.

네이버 카페가 허용하지 않는 일부 서식은 붙여넣기 과정에서 제거될 수 있습니다.
