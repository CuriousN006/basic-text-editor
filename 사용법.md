# 오프라인 리치 라이터 사용법

`index.html`은 설치 없이 Edge 또는 Chrome으로 실행하는 로컬 문서 편집기입니다.

## 실행

1. `index.html` 파일을 더블클릭합니다.
2. 연결 프로그램을 묻는 경우 Edge 또는 Chrome을 선택합니다.
3. 상단의 **파일 열기**로 `.md`, `.txt`, `.html` 파일을 불러옵니다.

인터넷 연결, 로그인, 서버, Node.js가 필요하지 않습니다.

## 네이버 카페에 복사

1. 편집을 마친 뒤 **전체 서식 복사**를 누릅니다.
2. 네이버 카페 글쓰기 창에서 `Ctrl+V`를 누릅니다.
3. 네이버가 일부 서식을 제거할 수 있으므로 게시 전 결과를 확인합니다.

## 다른 곳의 글 붙여넣기

- `Ctrl+V`: 외부의 굵게·색상·글꼴 같은 서식을 가져오지 않고, 일반 텍스트의 줄바꿈·빈 줄·탭을 항상 그대로 붙여넣습니다. 다만 글자를 선택한 상태에서 URL을 붙이면 선택한 글자에 링크가 걸립니다.
- `Ctrl+Shift+V`: 같은 방식으로 원문의 줄 구조를 보존하되 URL 자동 링크도 만들지 않고 URL 자체를 붙여넣습니다.

화면 폭 때문에 자동으로 꺾여 보이는 줄은 실제 줄바꿈이 아니므로 복사되지 않습니다. 원본에서 `Enter`나 `Shift+Enter`로 만든 줄바꿈만 보존할 수 있습니다.

## Word와 비슷하게 링크 넣기

1. 브라우저 주소창에서 URL을 `Ctrl+C`로 복사합니다.
2. 편집기에서 링크를 걸 글자를 선택합니다.
3. 그대로 `Ctrl+V`를 누릅니다.

또는 글자를 선택하고 `Ctrl+K`를 누른 뒤 URL을 붙여 넣습니다.

링크가 걸린 글자 위에 마우스를 올리면 연결 주소가 작은 말풍선으로 표시됩니다. 링크를 실제로 열려면 Word처럼 `Ctrl`을 누른 상태에서 링크를 클릭합니다.

## 서식 제거

글자를 선택하고 **서식 제거**를 누르면 굵게, 기울임, 링크뿐 아니라 제목 서식도 함께 제거되어 기본 본문으로 바뀝니다. 문서 내용을 전부 지우면 다음 입력부터 자동으로 기본 본문 서식이 적용됩니다.

커서가 굵은 글자 안에 있거나 다음 입력이 굵게 설정된 동안에는 상단의 **B** 버튼이 파란색으로 선택 표시됩니다. 일반 글자로 이동하거나 굵게를 끄면 선택 표시도 꺼집니다.

## 조사까지 자동으로 찾아 바꾸기

`Ctrl+H`로 찾아 바꾸기 창을 열면 **조사 자동 교정**이 기본으로 켜집니다. 단어 바로 뒤의 조사를 새 단어의 받침에 맞춰 함께 바꿉니다.

- `사과를`에서 `사과` → `책`: `책을`
- `책은`에서 `책` → `바나나`: `바나나는`
- `책으로`에서 `책` → `길`: `길로` (`ㄹ` 받침 예외 적용)

지원하는 조사는 `은/는`, `이/가`, `을/를`, `과/와`, `으로/로`, `이랑/랑`, `이나/나`, `이에요/예요`와 `으로서/로서`, `으로써/로써`, `으로부터/로부터`입니다. 입력한 글자만 그대로 바꾸려면 **조사 자동 교정**의 체크를 끕니다.

## 문단과 줄바꿈 찾아 바꾸기

Word처럼 찾기 또는 바꾸기 칸에서 다음 특수기호를 사용할 수 있습니다.

| 특수기호 | 의미 |
|---|---|
| `^p` | 문단 나눔 (`Enter`) |
| `^l` | 같은 문단 안의 줄바꿈 (`Shift+Enter`) |
| `^t` | 탭 |
| `^^` | `^` 문자 자체 |

예를 들어 찾기에 `^p^p`, 바꾸기에 `^p`를 입력하고 **모두 바꾸기**를 누르면 연속된 빈 문단을 줄일 수 있습니다. 찾기에 `^p`, 바꾸기에 `^l`을 입력하면 서로 다른 문단을 같은 문단 안의 줄바꿈으로 바꿉니다.

특수기호는 대문자와 소문자를 모두 인식합니다. 실제 `^p`라는 글자를 찾으려면 `^^p`를 입력합니다.

## 주요 단축키

| 기능 | 단축키 |
|---|---|
| 굵게 | `Ctrl+B` |
| 기울임 | `Ctrl+I` |
| 밑줄 | `Ctrl+U` |
| 하이퍼링크 | `Ctrl+K` |
| 찾기 | `Ctrl+F` |
| 찾아 바꾸기 | `Ctrl+H` |
| HTML 저장 | `Ctrl+S` |

## 파일 형식

- **HTML 저장**: 서식을 가장 충실하게 보존합니다. 브라우저로 다시 열 수 있습니다.
- **MD 저장**: 다른 마크다운 프로그램과 공유할 때 사용합니다. 밑줄처럼 표준 마크다운에 없는 서식은 HTML 태그로 저장될 수 있습니다.
- **마크다운 열기**: 제목, 굵게, 기울임, 취소선, 링크, 목록, 인용문, 코드 블록과 빈 줄을 지원합니다.

## 주의사항

- 이 편집기는 브라우저의 리치 텍스트 기능을 사용합니다. 브라우저가 바뀌면 일부 동작이 조금 다를 수 있습니다.
- 네이버 카페 편집기가 허용하지 않는 색상, 간격 또는 특수 서식은 붙여넣을 때 제거될 수 있습니다.
- 중요한 문서는 작업 중간에 HTML로 저장해 두는 것이 안전합니다.
